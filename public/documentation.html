myMovies Document

This is an REST API for an application called “myMovies” that interacts with a database that stores data about different movies.
User's will be able to search and access all of their favorite movies, genre of movies, directors including being able to sign up, update their personal inforamtion, and create a list of their favorites movie.

Adding Users

/users

Method Type: POST

Description:This endpoint allows users to register. Endpoint returns a JSON object with few key values. For key "name" it returns string,for "favorite movies" array, and for an "id" string.

Query Parameters: None

Expected request data: JSON object with data about user. Required: "name"

Example:

{
"Name": "Brayan Mejia",
}

Expected response data: JSON object with data about user which includes automatic generated ID

Example:

{
"Name": "Brayan Mejia",
"Email": "Brayan Mejia@email.com,"
"Password": "password123,"
"Favorite Movies": [],
"id": "8a757bb9-2eec-46bf-86f0-443ff742cf3a"
}

Response status codes:

201: Created- New user has been added

400: Bad request- User already exists



Updating Usernames

/users/:id

Method Type: POST

Description: This endpoint allows users to update their username. Endpoint returns a JSON object with few key values. For key "name" it returns string, for "favorite movies" array, and for an "id" integer.

Query Parameters: :id,

Expected request data: JSON object with data about user.

Required: :id

Example:

localhost:8080/users/1


{
"Username": "newUsername"
}

Expected response data: JSON object with data about user which includes,"id" with integer value, "Name", and "favorite movies" with an array.

Example:

{
"Name": "Brayan Mejia",
"Email": "Brayan Mejia@email.com",
"Password": "password123",
"FavoriteMovies": [],
"id": 8a757bb9-2eec-46bf-86f0-443ff742cf3a,
}

Response status codes:

200: OK- New user has been added

500: Internal server error- Something went wrong



Adding a Movie to Favorites

/users/:id/movieTitle

Method Type: POST

Description:This endpoint allows users to add movie to their favorite list.Endpoint returns an JSON object with users data including updated favorite movies array.

Query Parameters: :id, :movieTitle

Expected request data: Required: :id, :movieTitle

Example:

localhost:8080/users/1/movies/New%20Movie

Expected response data: JSON object with updated list of movies in Favorite Movies array.

{ The Batman has been added to user's movies list }

Response status codes:

200: OK- Updated users array list

500: Internal server error- Something went wrong

Removing a Movie from Favorites

/users/:id/:movieTitle

Method Type: DELETE

Description: This endpoint allows users to remove movie from their list. Endpoint returns an JSON object with users data including updated favorite movies array.

Query Parameters: :id, :movieTitle

Expected request data: Required: /id, /movieTitle

Example:

http://localhost:8080/users/2/The%20Batman

Expected response data: A message indicating the movie has been added to the user's movie list.

Example:

{ The Batman has been removed from user: 2's Movies list }

Response status codes:

200: OK- Movie has been removed from users list!

500: Internal server error- Something went wrong



Deleting Users

/users/:id

Description: This endpoint allows users remove their account by their Username.Endpoint returns a confirmation message that users account has been deleted.

Query Parameters: :id,

Expected request data: Required: :id

Example:

localhost:8080/users/1

Expected response data: A confirmation message that users account has been deleted.

Example:

{ "user 1 has been deleted" }

400: Bad request- Users name was not found
